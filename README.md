# Album-Ranker
  A simple webpage where two user have the ability to select two albums every week, listen to both albums, and rank them at the end of each week. 

## Features
  - Two users
  - Weekly album selection (one album per user)
  - Album ranking (1-10)
  - Display average scores for the week

## Tech Stack 
  - Frontend: HTML, CSS, TypeScript
  - Backend: Python (Flask)
  - Database: SQLite

## Goal
Build a clean, minimal app to practice full-stack development and track weekly album rankings.
